# SiSePuede Project

## Project Structure
```
SiSePuede/
├── notebooks/          # Jupyter notebooks for analysis
├── data/
│   ├── raw/           # Original data files (not in Git)
│   └── processed/     # Cleaned data (not in Git)
├── output/
│   ├── figures/       # Generated charts
│   └── reports/       # Analysis reports
└── src/               # Python utility scripts
```

## Setup
1. Clone: `git clone https://github.com/Zonova-Tech/SiSePuede.git`
2. Environment: `conda create -n sisepuede python=3.11`
3. Activate: `conda activate sisepuede`
4. Install: `pip install -r requirements.txt`
5. Launch: `jupyter lab`

## Usage
1. Place raw data in `data/raw/`
2. Open notebooks in `notebooks/`
3. Run analysis
4. Results saved to `output/`
